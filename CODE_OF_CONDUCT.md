# Code of Conduct

This project adopts the
[Contributor Covenant, version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

By participating in this project — as a contributor, maintainer, or
community member — you agree to abide by its terms.

## Scope

This Code applies to all project spaces: the GitHub repository, issue
tracker, pull requests, discussions, and any other venue where the
project's maintainers represent it publicly.

## Enforcement

Please report concerns to the maintainers via GitHub's private reporting
channels (see [`SECURITY.md`](SECURITY.md) for the disclosure link,
which doubles as a private contact path until a dedicated address is
published).

The maintainers are responsible for interpreting and enforcing this
Code fairly and consistently, and will respect the privacy of reporters.
